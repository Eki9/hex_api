# Hex API

### Used tecchnologies:
- Node.js
- Express
- Mocha + Chai (Unit Tests)
- Supertest (Integration Tests)

### Clone the repository:

``` bash
git clone https://github.com/eki9/hex_api.git
cd hex_api
```

### Install dependencies:
``` bash
npm install
```

### Start server by running:

``` bash
npm start
```

### Run test:

``` bash
npm test
```
